Instructions: How to add Crypto Emergency to your MetaMask wallet

Add Network

We will tell you how to add Crypto Emergency to MetaMask for use CEM coin.

To add a CEM network, you need to unlock the MetaMask wallet and click on the account logo in the upper right corner.

Select the "Settings" item from the menu

Below we look for the "Networks" section and click on it.

Now you see a list of all added networks, click on the "Add network" button.

In the Add New Network form, fill in everything by pattern.

Network Name: Crypto Emergency

URL RPC: https://cemchain.com/

Chain ID: 193

Currency symbol: CEM

Block Explorer URL: https://cemscan.com

You can specify any network name. (Crypto Emergency)

After filling in all the fields, click "Save."

Congratulations, you added Crypto Emergency to your MetaMask and can now store CEM coins
